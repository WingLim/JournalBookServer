# JournalBook Server

A Serverside for [JournalBook](https://github.com/WingLim/JournalBook)

## Usage

1. Clone code
```bash
$ git clone https://github.com/WingLim/JournalBookServer.git
```

2. Install requirements
```bash
$ pip install -r requirements.txt
```

3. Edit config.py
```python
pwd = "yourtoken"
```

4. Run server
```bash
$ pyhon api.py
```
